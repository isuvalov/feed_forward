Сделана удобоворимая синхронизация

Максимальная расстройка для пилота получается что высчитывается так:
( 1/(Npilot*InterpolateRate*(1/Fd)) )* (((MULT_POROG/(2**div_value))-1)/(MULT_POROG/(2**div_value)))=
= ( 1/(64*4*(1/100e6)) ) * (((70/(2**4))-1)/(70/(2**4))) =~ 301.34 кГц

--------------
Вообще раньше было так:
  <Генератор пилотов>+<генератор данных>  --->> радио канал --->>  <Нахождение пилотов по корреляции>,<нахождение синхронизации>,<нахождение расстройки частоты>,<коррекция расстройки частоты>,<feed forward>

Теперь будет так:

(<Генератор пилотов>+<генератор пилотного sin>)/(<Генератор пилотов>+<генератор данных>)  --->> радио канал --->> <нахождние расстройки частоты по sin>,
<коррекция расстройки частоты>,<Нахождение пилотов по корреляции>,<нахождение синхронизации>,<нахождение сдвига на дробный такт "Gardner TED">,<коррекция дробного сдвига - "фильтр Фарроу">,<выбеливание передаточной функции канала - фильтр Винера с функцией Гадарга>/<feed forward>
